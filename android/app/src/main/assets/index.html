6마리 지참 → 3마리 선발(팀 프리뷰). 내 팀엔 도구·기술까지 설정해 실제 커버리지와 선공으로 최적 3마리를 추천합니다.

① 데이터 내보내기 / 가져오기

포켓몬 도감(이름·타입·스피드)과 빌드(도구·기술)를 함께 백업/복원해요. 다른 브라우저·기기로 옮길 때 사용하세요.

② 도감에서 붙여넣어 추가

포켓몬 도감 표를 드래그 복사(텍스트·HTML 모두 인식) → 붙여넣기 → 추가. 중복 자동 제거, 지역폼은 타입 꼬리표로 구분.

③ 목록에 없는 포켓몬 직접 추가

아직 목록에 없는 포켓몬을 이름·타입으로 직접 넣으세요. 스피드는 선택(넣으면 선공 계산에 반영).
