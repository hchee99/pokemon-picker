# 포켓몬 싱글 3선발 도우미

6마리 지참 → 3마리 선발(팀 프리뷰) 방식의 싱글 배틀에서, 내 팀과 상대 팀을 넣으면 타입 커버리지·선공·약점 노출을 계산해 **최적의 3마리 조합**을 추천하는 도구.

## 사용법
1. `index.html`을 브라우저로 열기 (더블클릭 — 인터넷 불필요)
2. 처음엔 포켓몬 데이터 채우기
   - **도감 붙여넣기**: 포켓몬 도감 표를 드래그 복사 → 붙여넣기 → 추가 (텍스트·HTML 모두 인식)
   - **직접 추가**: 목록에 없는 포켓몬은 이름·타입으로 입력
   - **가져오기**: 기존 JSON 데이터가 있으면 붙여넣거나 파일로 불러오기
3. (선택) 내 포켓몬에 ⚙로 도구·기술 타입 설정 → 실제 커버리지·선공 반영
4. 내 팀 6칸 + 상대 팀 6칸 입력 → **최적 3마리 조합 찾기**
   - 추천 카드를 클릭하면 상대별 처리 상세가 펼쳐짐
   - 자주 쓰는 팀은 프리셋으로 저장/불러오기
   - 메가진화는 한 배틀에 1마리 제한이 자동 적용됨

## 결과 지표
- **커버 X/6** — 상대를 2배↑로 때리는 수 (4배 개수 표시)
- **선공 커버** — 스피드로 앞서면서 2배로 때리는 상대 수
- **위험** — 상대 자속에 내 포켓몬이 2배로 맞는 매칭 수 (낮을수록 안전)

## 데이터
- 브라우저 localStorage에 자동 저장 (같은 브라우저에서 유지)
- 이름이 같고 타입이 다른 지역폼은 `윈디 (불꽃/바위)`처럼 자동 구분

## 한계
순수 타입 상성 기반. 실제 배틀은 기술 배치·도구·특성·스피드 실능(노력치·성격)에 따라 달라질 수 있음.
