# 포켓몬 싱글 3선발 도우미

6마리 지참 → 3마리 선발(팀 프리뷰) 방식의 싱글 배틀에서, 내 팀과 상대 팀을 넣으면 타입 커버리지·선공·약점 노출을 계산해 **최적의 3마리 조합**을 추천하는 도구.

브라우저만 있으면 되는 단일 HTML 파일이고, 서버·인터넷 없이 동작합니다. 모든 데이터는 브라우저 안(localStorage)에만 저장됩니다.

## 주요 기능
- **최적 3마리 추천** — 가능한 조합을 전부 계산해 커버리지·선공·약점 기준으로 순위화
- **도구·기술·스피드 버프 설정** — 내 포켓몬마다 지닌 물건 1개 + 기술 타입 4개 + 순풍·용춤 등 스피드 버프를 넣어 실제 커버리지·선공으로 계산
- **메가 1마리 제한** — 한 배틀에 메가진화 1마리 규칙을 추천에 자동 적용
- **팀 프리셋** — 자주 쓰는 내 팀 6마리를 이름 붙여 저장/불러오기
- **상세 설명** — 추천 카드를 클릭하면 상대별 처리(누가·무슨 타입으로·몇 배·선공/후공)를 표로
- **데이터 관리** — 도감 붙여넣기 / 직접 추가 / JSON 내보내기·가져오기

## 사용법
1. `index.html`을 브라우저로 열기 (더블클릭 — 인터넷 불필요)
2. 처음엔 포켓몬 데이터 채우기
   - **도감 붙여넣기**: 포켓몬 도감 표를 드래그 복사 → 붙여넣기 → 추가 (텍스트·HTML 모두 인식, 중복 자동 제거)
   - **직접 추가**: 목록에 없는 포켓몬은 이름·타입·(스피드) 직접 입력
   - **가져오기**: 기존 JSON이 있으면 붙여넣거나 파일로 불러오기
3. (선택) 내 포켓몬 슬롯의 **⚙** 버튼으로 도구·기술 타입·스피드 버프 설정
4. 내 팀 6칸 + 상대 팀 6칸에 이름 입력 (자동완성)
5. **최적 3마리 조합 찾기** → 추천 카드 확인, 클릭하면 상대별 상세가 펼쳐짐

## 결과 지표
- **커버 X/6** — 상대를 2배↑로 때리는 수 (4배는 별도 표시)
- **선공 커버** — 스피드로 앞서면서 2배↑로 때리는 상대 수 (구애스카프 ×1.5, 순풍·랭크업 등 스피드 버프 배율도 함께 반영)
- **위험** — 상대 자속에 내 포켓몬이 2배↑로 맞는 매칭 수 (낮을수록 안전)

추천 점수 = `커버 × 100 + 선공 커버 × 10 + 4배 × 4 − 위험 × 6` (높은 순 정렬).

## 계산 규칙
- 기술 타입을 설정하면 그 타입들로 커버를, 미설정 시 자속 타입으로 커버를 계산
- 도구는 스피드에 영향을 주는 것(구애스카프 등)만 계산에 반영, 나머지는 표시용
- 스피드 버프(순풍 ×2, 용춤 등 랭크+1 ×1.5, 랭크+2~+6 등)는 배율로 스피드 계산에 반영되며, 도구 효과와 중첩 적용됨
- 메가진화(이름이 `메가`로 시작, 단 `메가니움`은 제외)는 3마리 중 1마리까지만
- 이름이 같고 타입이 다른 지역폼은 `윈디 (불꽃/바위)`처럼 자동 구분 저장

## 데이터 저장
- 도감·빌드·프리셋 모두 브라우저 localStorage에 자동 저장 (같은 브라우저에서 유지)
- **내보내기/가져오기 JSON에 도감 + 빌드 포함** → 다른 기기·브라우저로 옮길 때 사용
- 데이터는 어떤 서버로도 전송되지 않고 기기 안에만 있음

## 한계
순수 타입 상성 기반입니다. 실제 배틀은 기술 위력·특성(부유 등)·스피드 실능(노력치·성격)·랭크업 등에 따라 달라질 수 있으니 선발 참고용으로 사용하세요.
